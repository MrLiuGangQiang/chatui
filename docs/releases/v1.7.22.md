# ChatUI v1.7.22

Released on July 23, 2026.

## Fixes

- Streaming Markdown now keeps native and shorthand details blocks atomic while hiding incomplete Markdown control markers from users.
- Completed streamed Markdown is reconciled through one canonical parse, so tables, lists, and multiline blocks match the result shown after a page refresh.
- Large Markdown responses are parsed as one complete document, mounted offscreen in batches, and swapped into view atomically.
- Code-copy and Mermaid source/render controls now keep stable geometry while using a visible background and subtle rounded-corner interaction.

## Verification

- Added regression coverage for details streaming, canonical final rendering, large Markdown, and code/Mermaid action controls.
- `npm run check` passes before release.
