# ChatUI v1.7.25

## 会话附件草稿隔离

- 未发送的图片和文件现在归属于发起上传的会话；切换到其他会话或新建会话时，不会再显示前一个会话的附件。
- 返回原会话会恢复其未发送附件草稿，与输入文字草稿保持一致。
- 上传尚未完成时切换会话，处理结果仍只会写回原会话，不会泄漏到当前会话。
- 清空附件或删除会话会使正在处理的旧上传结果失效，并同步清理上传进度、运行时草稿和关联的图片缓存引用。

## 图片编辑路由稳定性

- 当前上传图片的临时附件 ID 现可安全归并到稳定图片引用 ID，避免图片编辑意图已识别但无法绑定上传目标。
- 归并仅适用于本轮、同序号图片；不相关附件 ID 仍会被拒绝，保持资源选择边界。
