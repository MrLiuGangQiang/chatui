# ChatUI v1.7.4

Released on July 20, 2026.

## Fixes

- Prevented macOS input methods from sending a message when Enter is used to confirm an IME candidate; normal Enter-to-send and Shift+Enter line breaks are unchanged.
- Stopped an empty visible image-model selection from reviving a stale stored model such as `gpt-image-2`.
- Centered transient toast messages, including “已清除所有会话”, over the active chat content region instead of the full browser viewport.

## Verification

- Added regression coverage for active IME composition, legacy IME key code 229, Safari-style Enter events immediately after composition ends, and normal Enter submission.
- Added regression coverage ensuring empty visible chat and image model selections override stale stored values.
- Verified toast alignment against the chat/composer center in expanded desktop, collapsed desktop, and mobile layouts.
- Verified the IME event sequence in Microsoft Edge and confirmed the served static bundle contains the updated guard.
- `npm run check` passes with 368 tests.
