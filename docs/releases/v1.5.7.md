# ChatUI v1.5.7

## Welcome experience

- Redesigned the empty-chat welcome page with a cleaner enterprise card layout.
- Presents ChatUI's core capabilities as concise cards for intelligent chat, deep reasoning, file understanding, and image creation.
- Improved desktop and mobile spacing so the welcome content remains clear above the composer.

## Usage guidance

- Clarifies that AI output should be verified and sensitive data must not be uploaded.
- Explains that chat history is stored only in the current browser and should be backed up before upgrades.
- States that company AI is for work use only and that account bans or Token usage caused by noncompliant use are the user's responsibility.

## Verification

- Updated welcome-page regression coverage and browser cache versions.
- Verified the redesigned page at desktop and mobile viewport sizes.
- Docker deployments can upgrade to the `1.5.7` or `v1.5.7` image tag.
