# ChatUI v1.7.20

Released on July 23, 2026.

## Improvements

- Markdown tables now render as a native live table while an answer streams, without briefly exposing table pipes or divider markers.
- The table renderer processes only newly completed rows, batches rapid updates, and keeps the final response on the normal Markdown rendering path.
- Table alignment, escaped pipe handling, and horizontal scrolling are consistent between streaming, large-message previews, and completed responses.

## Verification

- Added regression coverage for live table candidates, column alignment, escaped pipes, batched updates, and large Markdown previews.
- `npm run check` passes before release.
