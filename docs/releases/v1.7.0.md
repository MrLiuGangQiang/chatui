# ChatUI v1.7.0

## Smarter, stricter task routing

- Replaces the previous model-route shape with one strict `task_contract.v3` protocol and no legacy adapter path.
- Uses a single `operation` field to derive the runtime API and mode, preventing contradictory intent and execution fields.
- Adds deterministic structured patches for follow-ups, corrections, image edits, and reference-image generation.
- Prevents unrelated new requests from inheriting historical image prompts or resources.
- Applies valid AI review corrections even when the revised confidence score is lower than the first pass.

## Large-message input protection

- Prevents oversized paste, drop, and text insertion before the browser mutates the composer.
- Caps a single text message at 120,000 characters and recommends file upload or segmented messages for larger content.
- Skips synchronous `scrollHeight` measurement for large prompts, avoiding the forced-layout freeze that previously happened before sending.
- Removes duplicate textarea resize listeners and keeps a final submission guard for restored or programmatic input.

## Verification

- Added regression coverage for strict route contracts, context isolation, structured patch composition, pre-insertion size checks, and large-input layout behavior.
- Verified the complete project check suite with 341 passing tests.
- Docker deployments can upgrade to the `1.7.0` or `v1.7.0` image tag.
