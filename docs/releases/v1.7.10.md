# ChatUI v1.7.10

Released on July 20, 2026.

## Fixes

- Fixed the first intent-routing request after switching a session model so a transient failure retries the selected session model instead of incorrectly reporting that both route models are unavailable.
- Routed invalid or unparseable primary intent results through the same fallback attempt rather than failing immediately.
- Renamed the welcome-page workspace label from `AOZHE AI WORKSPACE` to `AUTHINE AI WORKSPACE`.
- Refreshed the route workflow and static-bundle cache versions so browsers receive the fix immediately.

## Verification

- Added regression coverage for same-model fallback retry after a model switch and for invalid primary route responses.
- `npm run check` passes with 373 automated tests.
