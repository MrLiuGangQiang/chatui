# ChatUI v1.7.11

Released on July 20, 2026.

## Fixes

- Restored the stable v1.7.8 intent-routing fallback behavior so an unavailable model is not retried through an identical duplicate request.
- Made invalid intent-contract responses fail deterministically instead of silently switching models.
- Ensured pending clarification classification uses the effective route model for the target session.
- Ensured session and explicit intent-recognition model changes are read on the next submission without retaining a stale model selection.
- Prevented session, global chat, and intent-recognition model changes while tasks are running, avoiding mixed-model execution within one submission.
- Restored blocked global model selections in the configuration UI and kept the previously persisted models unchanged.

## Verification

- Added regression coverage for session-model switching, explicit route-model switching, follow mode, distinct-model fallback, invalid contracts, pending clarification routing, and busy-task model guards.
- `npm run check` passes with 377 automated tests.
